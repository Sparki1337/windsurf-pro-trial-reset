## Security Best Practices

When using Windsurf Reset Tool:

1. **Download from official sources only**
   - Only download from the official GitHub repository
   - Verify checksums when available

2. **Review permissions**
   - The tool only requires access to Windsurf configuration files
   - No network access is required

3. **Backup your data**
   - Always create backups before resetting IDs
   - Keep important configuration files safe

4. **Keep software updated**
   - Check for updates regularly

## Privacy

This tool:
- Does NOT collect any telemetry or usage data
- Does NOT send any information over the network
- Only modifies local Windsurf configuration files
- Creates local backup files only

All operations are performed locally on your machine.
